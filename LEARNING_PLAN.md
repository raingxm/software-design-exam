# 软件设计师（中级）复习计划 - 2026年下半年

## 学习理念
- **结构化思维**：利用 Markdown 和 Mermaid 图表将知识点体系化。
- **以练促学**：通过 TypeScript 实现算法和设计模式，加深理解。
- **视觉化学习**：所有架构、流程和关系图均使用 Mermaid 绘制。
- **高效迭代**：关注重点和难点，不纠结于琐碎知识点。

## 复习阶段
1. **第一阶段：夯实基础（计算机组成、OS、网络、数据库）**
2. **第二阶段：核心突破（软件工程、结构化/面向对象分析与设计、设计模式）**
3. **第三阶段：专项攻克（算法与数据结构、法律法规、标准化）**
4. **第四阶段：冲刺实战（历年真题演练、查漏补缺）**

## 目录结构
- `notes/`: 核心知识点笔记
- `exercises/`: 算法与设计模式的代码实现
- `practice/`: 历年真题分析
- `syllabus/`: 大纲与进度跟踪
