---

## 口令

```text
依次 / 经…后输出 / 流水线把数据逐步变掉  → 变换
按类型 / 分别交给 / 分发到不同路径      → 事务
多个加工排成一条线仍是变换；按类型分叉才是事务
```

---

## 两种中心（人话）

| | **变换型** | **事务型** |
|--|------------|------------|
| 像什么 | 加工厂流水线 | 分拣台 / 交换机 |
| DFD 直觉 | 汇入 → 中心加工 → 输出 | 中心后扇出多条互斥路径 |
| 结构图 | 主控下挂：输入支 · 变换支 · 输出支 | 主控下挂：接收 · 调度 · 多个事务模块 |

```text
变换：          事务：
    主控              主控
 /   |   \          /    \
输入 变换 输出     接收   调度
                      / | \
                   A   B   C
```

---

## 已过关例题

| 描述 | 答 |
|------|----|
| 滤波 → 特征提取 → 分类 → 告警级别 | 变换 |
| 工单按报修/咨询/投诉转到不同子系统 | 事务 |
| 地址规范化 → 运费 → 库存预扣 → 发货单 | 变换 |
| 报文按类型交给登录/支付/推送 | 事务 |
| 去噪 → 边缘检测 → 轮廓坐标 | 变换 |

---

## 考试要什么 / 不要什么

- **要**：给场景或简图，选变换还是事务  
- **不要**：默画完整结构大图（认形态即可）  
- 与 DFD「缺流」无关：一个是补水管图，一个是选拆模块策略
